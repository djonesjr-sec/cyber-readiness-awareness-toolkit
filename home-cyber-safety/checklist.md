# Home Cyber Safety Checklist

## Wi-Fi and Router Security

- [ ] Change the default router admin password
- [ ] Use WPA2/WPA3 security
- [ ] Disable WPS if not needed
- [ ] Review connected devices monthly
- [ ] Update router firmware regularly
- [ ] Use a strong Wi-Fi password

## Camera and Smart Device Safety

- [ ] Use a separate IoT network for cameras and smart devices
- [ ] Keep visitor devices on a separate guest network
- [ ] Use strong passwords for camera apps
- [ ] Enable MFA on camera or smart home accounts
- [ ] Avoid exposing cameras directly to the internet
- [ ] Keep smart devices updated

## Password Safety

- [ ] Use a password manager
- [ ] Avoid reusing passwords
- [ ] Use long passwords or passphrases

## Multi-Factor Authentication

- [ ] Enable MFA on email accounts
- [ ] Enable MFA on banking accounts
- [ ] Save backup recovery codes securely

## Email and Scam Awareness

- [ ] Be cautious with unexpected links
- [ ] Verify payment requests through another method
- [ ] Watch for fake delivery messages
- [ ] Watch for QR code scams

## Backups and Recovery

- [ ] Back up important photos and documents
- [ ] Verify recovery email and phone numbers
- [ ] Store important records securely
